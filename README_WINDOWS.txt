ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА НА WINDOWS
===============================================

Этот проект представляет собой библиотеку математических классов с геометрическими фигурами.

ВАЖНО: Проект теперь КРОСС-ПЛАТФОРМЕННЫЙ! 
Один и тот же код компилируется и работает на Windows, macOS и Linux.

ТРЕБОВАНИЯ:
-----------
- Windows 7/8/10/11
- Компилятор C++ (один из):
  * MinGW-w64 (рекомендуется)
  * Visual Studio 2017 или новее
  * Dev-C++
  * Code::Blocks с MinGW

БЫСТРЫЙ ЗАПУСК:
---------------
1. Убедитесь, что у вас установлен компилятор C++
2. Откройте командную строку (cmd) в папке с проектом
3. Запустите: compile_windows.bat
4. Программа автоматически скомпилируется и запустится

РУЧНАЯ КОМПИЛЯЦИЯ:
------------------

Если у вас MinGW:
g++ -std=c++17 -I. Source.cpp src/Figure.cpp src/Point.cpp src/Real.cpp src/Vector.cpp src/Fraction.cpp src/Integer.cpp -o test_program.exe

Если у вас Visual Studio (из Developer Command Prompt):
cl /EHsc /std:c++17 /I. Source.cpp src/Figure.cpp src/Point.cpp src/Real.cpp src/Vector.cpp src/Fraction.cpp src/Integer.cpp

ЗАПУСК:
-------
После компиляции запустите: test_program.exe

СОСТАВ ПРОЕКТА:
---------------
- Source.cpp - главный файл программы
- hd\ - папка с заголовочными файлами (.hpp)
- src\ - папка с исходными файлами (.cpp)
- compile_windows.bat - автоматическая компиляция

ДЕМОНСТРАЦИЯ:
-------------
Программа демонстрирует работу с тремя типами фигур:
- Треугольник (вычисление площади и периметра)
- Круг (вычисление площади и длины окружности)
- Параллелограмм (вычисление площади и периметра)

Также показывается полиморфная работа с фигурами через базовый класс.

ВОЗМОЖНЫЕ ПРОБЛЕМЫ:
-------------------
1. Если не отображаются русские символы:
   - Убедитесь, что консоль поддерживает UTF-8
   - В новых версиях Windows это работает автоматически

2. Если компилятор не найден:
   - Установите MinGW или Visual Studio
   - Добавьте путь к компилятору в переменную PATH

3. Если ошибки компиляции:
   - Проверьте, что все файлы на месте
   - Убедитесь, что используется C++17 стандарт
